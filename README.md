# Device Lens Testing

Welcome to the Service Deployment team's repository for machine information and benchmark scores.

## About

The information contained in this repo is an aggregate of general machine data combined with benchmark data using the following tests:
- PAT: ArcGIS Performance Assessment Tool
- Cinebench: GPU, Multi-Core, Single-Core Scores
- Blender: 3 Tier Image Sampling rates

## Tools

[Datavzrd](datavzrd.github.io) is a low-code interactive tool that uses yaml to stucture .csv files into interactive browser-based reports for tabular datasets

### Dependencies

- Conda
- Python

### Accepted File types

- CSV
- TSV
- JSON
- Parquet
